# Timetable Scheduler

A prototype web application to generate timetables for educational institutions.

## Live Demo

Try it online: [https://timetable-api.onrender.com](https://timetable-api.onrender.com)

## Features

- Add / update / delete rooms, faculties, subjects, and batches
- Generate randomized weekly timetable (5 days Ã— 6 slots)
- View timetable per batch

## Tech Stack

- Backend: FastAPI + SQLite
- Frontend: React + Axios
